# Friend Starter Tasks

Hand these out to Koa, Naren, Lauren, or Jian after the presentation. Each file is one job from slide 10 of the deck. When a friend says "I pick X", open the matching file and send them that text.

**Friends can pick more than one job.** Koa picked Bug Hunter + Tester. Naren picked Hype Person + Designer. That's totally fine — just send them both starter tasks.

Each starter task is designed to:
- Take 15 minutes or less for the friend
- Have a clear "done" definition
- Need zero coding knowledge
- Produce something Viraaj can actually use in the next release

## Workflow

1. **After the talk**, ask each friend which job sounds fun
2. **Open the matching file** here and copy the text
3. **Send it to them** however is easiest (text, DM, email, in person)
4. **Set a loose deadline** — "when you have 15 min" is fine
5. **When they send back their work**, open an issue on GitHub or add it to `tasks/contributions.md` with credit

Each task has two sections:
- **What to send them** — the actual task, friendly and clear
- **What Viraaj does with the result** — how to actually use what they give you

## Future tasks

Once a friend completes their first starter task, you can give them something bigger from that same category. Keep a mental list of things you'd love their help with.
